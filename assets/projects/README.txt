Taruh foto/screenshot projek di folder ini dengan nama PERSIS seperti berikut:

  weather-app.jpg     -> Weather App
  todo-app.jpg        -> To-Do List App
  kalkulator.jpg      -> Kalkulator Interaktif
  restoran.jpg        -> Landing Page Restoran
  notes-app.jpg       -> Catatan Digital (Notes App)
  python-scraper.jpg  -> Python Data Scraper

Tips:
- Ukuran disarankan rasio mendatar (landscape), misal 600x350 px.
- Format .jpg. Kalau mau pakai .png, ubah juga ekstensi di index.html.
- Kalau file belum ada, card otomatis tampil pakai emoji (fallback). Aman.
